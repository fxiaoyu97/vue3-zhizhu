在`resful API`中每个网址代表一种资源。

## 一、HTTP动词

+ GET（SELECT）：从服务器取出资源（一项或多项）
+  POST（CREATE）：在服务器新建一个资源
+ PUT （UPDATE）：在服务器更新资源
+ PATCH（UPDATE）：在服务器更新资源
+ DELETE （DELETE）：从服务器删除资源

## 二、常见状态码

HTTP 响应代码：https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Status

响应分为五类：

+ 信息响应(100–199)
+ 成功响应(200–299)
+ 重定向(300–399)
+ 客户端错误(400–499)
+ 服务器错误 (500–599)
